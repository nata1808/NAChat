# 🤖 NAChat - Mi Asistente de IA Personal

**NAChat** es un asistente de inteligencia artificial ultrarrápido que corre en Groq Cloud. Tiene memoria, busca en internet con enlaces reales y aprende de tus archivos PDF o TXT.

## ✨ Características
- ⚡ **Ultra rápido** gracias a Groq (respuestas en 1-2 segundos)
- 🧠 **Memoria de conversación** (recuerda todo lo que hablas)
- 🌐 **Búsqueda web automática** con enlaces clickeables (YouTube, noticias, etc.)
- 📁 **Carga de documentos** (PDF y TXT) para que la IA aprenda de tus datos
- 🏢 **Identidad personalizada**: creada por Natanael Ferrer el 10 de junio de 2026 en Bogotá, Colombia para Pevaar Software Factory S.A.S.

## 🚀 Cómo usarla
1. Escribe tu pregunta en el chat.
2. Activa la búsqueda web (opcional) o la IA lo hará automáticamente.
3. Sube archivos para que aprenda de tu información personal.

## 🛠️ Tecnologías
- Python + Streamlit
- Groq Cloud (LLM)
- DuckDuckGo Search (búsqueda web)
- PyPDF2 (lectura de PDFs)
